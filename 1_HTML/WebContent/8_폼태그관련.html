폼 관련 태그

폼 관련 태그는 html에서 사용자가 입력할 수 있는 양식을 제공하고 form 태그는 입력한 정보를 서버로 넘기는 역할
- action : 폼에 입력된 값들을 전송받을 서버의 클래스명/페이지명/url명 입력
   action 지정하지 않으면 현재 페이지로 옴
- method : get/post
   get 방식 : 기본 방식, url 뒤에 서버에 보낼 데이터가 노출됨, 데이터 크기 제한 o
   post 방식 : 기본 방식이 아니라 명시해야 함, url에 서버에 보낼 데이터를 노출시키지 않음, 데이터 크기 제한 x
